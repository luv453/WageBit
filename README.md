# WageBit

WageBit is a local-first Android dashboard for converting Bitcoin savings into stored work time.

The app calculates:

- Real hourly wage: `(monthly income - monthly expenses) / monthly work hours`
- Current Bitcoin value from user-entered BTC held and BTC price
- Work hours, work days, and work months represented by the Bitcoin stack
- A freedom date based on target expense years, DCA amount, DCA cadence, and BTC CAGR
- Inflation-adjusted expense goals using a user-entered cost-of-living increase
- Current age at the projected freedom date
- BTC time efficiency against total worked time since the earliest acquired BTC lot
- Goal progress plus the BTC price needed for freedom today
- A growth comparison chart for BTC, S&P 500, Nasdaq-100, and gold using the same DCA plan
- Optional hiding of total Bitcoin value throughout the app
- Editable dashboard language for the time capsule and freedom labels

## Why Local Input?

WageBit does not request network access, fetch prices, use analytics, or send financial data anywhere. Users enter their own current BTC price, BTC lots, acquired dates, DCA plan, and retirement assumptions.
